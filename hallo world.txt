git status : untuk mengecek status kondisi local repositorynya

1. git init
	inisialisasi folder sebagai repository
2. git add .
	untuk menambahkan file ke staging area
3. git commit -m "pesan komit"
	memberikan pesan komit dari versionnya
4. kalau repo baru?
   git remote add origin [alamat gitnya]
   git branch -M main
5. git push -u [namaremote] [namabranch]
	git push -u origin main

ada perubahan kode / file
1. save file
2. add file
	git add .
3. commit
	git commit -m "pesan komit lainnya"
4. push
	git push -u origin main
